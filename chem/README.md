## How to use this project:
- Https: `$ git clone https://github.com/1chooo/ATM-Chem.git`

- ssh: `$ git clone git@github.com:1chooo/ATM-Chem.git`

- CLI: `$ git clone gh repo clone 1chooo/ATM-Chem`
